Pada episode 6 mempelajari tentang lifecycle component yang ada di react, yang dimana terdiri dari :
--COMPONENT MOUNTING => merupakan komponen ketika ditampilkan
--COMPONENT UPDATING => komponen ketika dilakukan update data lalu kemudian ditampilkan
--COMPONENT UNMOUNTIMG => komponen ketika menghilang dari tampillan

lalu untuk mempelajari hal tersebut harus menggunakan libraries dari react-hook yaitu menggunakan useEffect
-useEffect sendiri merupakan fitur React-hook yang digunakan untuk mengatur efek samping atau side effect pada 
 komponen fungsional, seperti memuat data dari server atau mengubah judul halaman. useEffect akan dijalankan 
 setiap kali komponen dirender ulang.


lalu pada episode 6 juga mempelajari kegunaan React.StrictMode yang dimana digunakan untuk membantu developer
untuk mengecek atau mengatasi masalah potensial (coding) pada pembuatan aplikasi. sehingga dengan menggunakan 
React.StrictMode, developer dapat mengidentifikasi dan memperbaiki masalah potensial pada aplikasi React sejak dini.

PENTING DIINGAT bahwa <React.StrictMode> sebaiknya hanya digunakan pada tahap pengembangan (development) 
dan tidak digunakan pada aplikasi yang sudah berjalan di produksi (production).
